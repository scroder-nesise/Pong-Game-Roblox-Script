This is a client sided script. What's a Client sided script? 

In Roblox, a client-side script (historically known as a LocalScript) is a script that executes locally on a player's machine - such as their laptop, phone, or console, rather than on Roblox's central servers.

Because each player has their own device, every player in a game runs their own unique copy of your client-side scripts. 

What's a script? 

In Roblox, a script is a container for code that tells the game how to behave. It acts as the "brain" behind every action - from making a part spin to managing a player's inventory.

How to use it?
